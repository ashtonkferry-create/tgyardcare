Create TG-90 as a daily cron that sends fertilizer treatment reminders 7 days before each scheduled step.

Flow:
1. Schedule Trigger (daily 9:30 AM CT = 14:30 UTC)
2. Query fertilizer_schedule WHERE scheduled_date = NOW() + 7 days AND reminder_sent_at IS NULL
3. For each result: Check SMS consent -> IF consent: Send reminder SMS + Log -> THEN: Send reminder Email -> PATCH reminder_sent_at. IF no consent: Send email only -> PATCH reminder_sent_at
